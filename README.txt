B-COSFIRE: trainable filters for detection of curvilinear and elongated patterns in images

Originally proposed for:
- Delineation of blood vessels in retinal images using trainable B-COSFIRE filters

To run a demonstration:
1. Run the script BeforeUsing.m
2. Read the help contents of the script called BCOSFIRE.m
3. Use the example application in the file ExampleBloodVesselSegmentation.m

NOTE: The software requires a working MEX compiler.
